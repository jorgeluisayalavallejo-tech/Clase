# Sabor Casero

Este proyecto es una landing page sencilla para un restaurante llamado **Sabor Casero**.

La pagina fue creada con HTML, CSS, JavaScript y Bootstrap 5 por CDN. Incluye una barra de navegacion responsiva, un carousel principal, cards de platos destacados, modales con informacion adicional, una seccion de promociones, formulario de contacto y footer.

El diseño usa colores calidos y una estructura responsiva para que se vea bien en computadores, tablets y celulares.
